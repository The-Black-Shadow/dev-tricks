# Dev Tips and Tricks

A collection of open-source scripts, tips, and tricks for developers. Each feature is organized into its own folder with detailed documentation.

## Features

- [Google Cloud Text-to-Speech Script](./gcp-text-to-speech/README.md) - A script to generate custom audio using Google Cloud TTS.
- [Move Xcode & Android Cache to External Drive](./move-dev-cache-external/README.md) - A safe guide on reducing macOS internal SSD usage by moving heavy development IDE caches using symlinks (optimized for Flutter development for Android and iOS). Now includes an automated, interactive `.command` script for easy setup.
